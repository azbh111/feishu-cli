## 问题描述

```markdown
- [ ] 父任务
    - [ ] 子任务 1
    - [ ] 子任务 2
```

- **导入**：Todo 列表项的嵌套子列表被丢弃（`convertListItem` 直接返回，未收集 Children）
- **导出**：Todo 块类型未加入子块收集逻辑（`BlockTypeTodo` 缺失），子块被当作独立顶层块重复输出

## 改动内容

- 导入侧：3 处 `TaskCheckBox` 分支调用 `collectNestedChildren` 收集嵌套列表；`extractTextElementsSkipCheckbox` 跳过 `ast.List` 节点防止文本泄漏
- 导出侧：`NewBlockToMarkdown` 的子块收集 case 新增 `BlockTypeTodo`；新增 `convertTodoWithDepth` 替代原 `convertTodo`，支持 indent/depth 参数和递归
