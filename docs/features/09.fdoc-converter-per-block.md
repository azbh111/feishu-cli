# 09. ConvertPerBlock 按块拆分导出方法

- **分类**: fdoc
- **提交官方**: 否
- **类型**: feat

## 变更文件

| 文件 | 改动说明 |
|------|---------|
| `internal/converter/block_to_markdown.go` | 新增 `ConvertPerBlock()` 方法，返回 `[]BlockMarkdown`（每个元素含 BlockID + Markdown 文本） |

## 说明

fdoc push 差量更新需要将云端文档按块拆分为 Markdown 片段，与本地 Markdown 片段逐块比对。`ConvertPerBlock` 复用 `Convert()` 的转换逻辑，但返回值按 BlockID 拆分。
