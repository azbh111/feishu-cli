# 13. 语义树 diff 引擎

- **分类**: fdoc
- **提交官方**: 否
- **类型**: feat

## 变更文件

| 文件 | 改动说明 |
|------|---------|
| `internal/diff/doctree.go` | 核心类型 `DocNode`、`NodeType`、`NodeProps`、`Equal()`、`CanUpdate()` |
| `internal/diff/block_to_tree.go` | 飞书 Block 数组 → DocNode 语义树 |
| `internal/diff/md_to_tree.go` | Markdown → DocNode 语义树（通过 converter 中转） |
| `internal/diff/tree_diff.go` | LCS diff 算法：`DiffDocNodes()`、`mergeUpdates()` |
| `internal/diff/index.go` | `TreeDiffOp` 操作类型、`MergeDeleteRanges()`、`computeInsertPositions()` |
| `internal/diff/executor.go` | 执行计划生成：Update/Delete/Insert 操作、`HasChanges()`、`Summary()` |
| `internal/diff/*_test.go` | 7 个测试文件，覆盖所有模块 |

## 说明

fdoc push 的核心引擎。将本地 Markdown 和云端文档分别转换为语义树，通过 LCS 算法对比差异，生成最小化的 Update/Delete/Insert 操作序列，实现增量更新而非全量替换。

**仅被 `fdoc/cmd/push.go` 引用**，feishu-cli 的 cmd 不依赖此包。

## 依赖

- seq 09（ConvertPerBlock）：diff 引擎依赖按块拆分的导出能力
- `internal/converter`：依赖块类型常量和转换函数
