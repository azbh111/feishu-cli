# 11. frontmatter 包 + FolderID 配置

- **分类**: fdoc
- **提交官方**: 否
- **类型**: feat

## 变更文件

| 文件 | 改动说明 |
|------|---------|
| `internal/frontmatter/frontmatter.go` | 新包，95 行。`Parse()` 解析 YAML front matter（document_id, folder_id, user_id, url）；`Write()` 回写 front matter |
| `internal/config/config.go` | 新增 `FolderID` 字段（默认空，环境变量 `FEISHU_FOLDER_ID`） |

## 说明

- `frontmatter` 包：fdoc push/pull 的核心机制，Markdown 文件通过 front matter 记录关联的云端文档 ID，实现"文件即文档"的双向同步
- `FolderID` 配置：fdoc push 首次创建文档时的默认目标文件夹

两者仅被 fdoc 使用，feishu-cli 的 cmd 不依赖。

## 依赖

无。
