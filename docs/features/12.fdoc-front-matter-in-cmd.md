# 12. import/export 命令支持 front matter

- **分类**: fdoc
- **提交官方**: 否
- **类型**: feat

## 变更文件

| 文件 | 改动说明 |
|------|---------|
| `cmd/import_markdown.go` | 新增 `parseFrontMatter()`、`writeFrontMatter()`；新增 `--sync` flag；从 front matter 读取 `document_id` 进入更新模式（清空文档再重建） |
| `cmd/export_markdown.go` | `Args` 从 `ExactArgs(1)` → `RangeArgs(0, 1)`；未传文档 ID 时从目标文件 front matter 读取 |
| `cmd/export_pipeline.go` | 新文件，导出 `ParseMarkdownSegments()` 供 fdoc 包调用 |

## 说明

这些改动让 feishu-cli 的 import/export 命令能识别 Markdown 文件中的 front matter 元数据，是 fdoc push/pull 的基础设施。但对于不使用 front matter 的用户，行为无变化（向后兼容）。

**不提交原因**：这些改动是 fdoc 工作流的一部分，独立提交给官方缺乏上下文。如果未来 fdoc 整体进入官方仓库，可一并提交。
