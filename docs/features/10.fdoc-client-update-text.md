# 10. UpdateBlockText 原地更新文本块

- **分类**: fdoc
- **提交官方**: 否
- **类型**: feat

## 变更文件

| 文件 | 改动说明 |
|------|---------|
| `internal/client/docx.go` | 新增 `UpdateBlockText()` 方法，通过 `UpdateBlock` API 原地更新文本类块内容（Text/Heading/Bullet/Ordered/Todo） |

## 说明

fdoc push 增量更新的核心能力：检测到某个块内容变化时，直接更新该块的 TextElements，而不是 Delete + Insert。减少 API 调用并保留块 ID 稳定性。

## 依赖

无。
