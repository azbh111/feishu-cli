# 05. 修复导入时 QuoteContainer 空块未清理

- **分类**: feishu-cli
- **提交官方**: 是
- **类型**: fix
- **PR 分支**: `fix/quotecontainer-empty-block`
- **PR 标题**: `fix(import): 清理 QuoteContainer 内多余空块`

## 变更文件

| 文件 | 改动说明 |
|------|---------|
| `cmd/import_markdown.go` | 空块清理条件从 `BlockTypeCallout` 扩展为 `BlockTypeCallout || BlockTypeQuoteContainer` |

## 问题描述

导入时，Callout 块内的多余空块会被自动清理，但 QuoteContainer（引用块）内的空块未被同等处理，导致引用块内多出一个空行。

## 改动内容

一行条件扩展：将 QuoteContainer 加入空块清理逻辑。

## 依赖

- seq 02（引用块修复）：逻辑上关联，建议合并提交或在 seq 02 之后提交。


**注意**：`import_markdown.go` 的 diff 混合了大量 fdoc 相关改动（front matter、--sync、Mermaid），只需提取这一个 hunk。
