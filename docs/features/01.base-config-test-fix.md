## 问题描述

`TestInit_DefaultValues` 会读取本地 `~/.feishu-cli/config.yaml`，如果用户机器上存在配置文件，默认值断言会失败。测试结果取决于运行环境，不可复现。

## 改动内容

在测试中将 `HOME` 环境变量临时替换为 `t.TempDir()`，确保不读取真实配置文件。

## 依赖

无，完全独立。
