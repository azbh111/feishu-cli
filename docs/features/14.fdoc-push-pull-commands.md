## 说明

独立的 `fdoc` 二进制工具，提供简化的双向同步命令：

- `fdoc push xxx.md`：本地 Markdown 推送到飞书云文档。首次创建文档，后续基于 diff 增量更新
- `fdoc pull xxx.md`：从云端拉取最新内容到本地 Markdown

`fdoc` 重度依赖 feishu-cli 的 internal 包（client、config、converter、diff、frontmatter、auth），是 feishu-cli 生态的上层工具。
